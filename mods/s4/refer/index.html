Refer is a mod for the game "The Sims 4"

Refer is a mod that allows sims to finally be talked about with pronouns that match their gender identity. Gender non-conforming sims can now easily change their preferred pronouns, sims can swap between the "male" and "female" pronouns, switch to using they / them, or select a "neopronoun" to be called by.

Note: This mod currently only works if your using the English version of the game.

Visit the issues section for important information!

Features

  • Easily change any sim's preferred pronouns at any time
  • Select from 5 possible non-binary pronoun sets, including 4 of the most common "neopronouns"

Questions and Answers

Q: Can I install this mod without having to start a new game?
A: Yes! You should be able to install this mod at any point in your play-through without issue.

Q: How easy is it to remove this mod? Will uninstalling it break my save game?
A: This mod can be easily removed by deleting all its files or using the mod's included uninstaller. Removing this mod mid play-through should cause no issues.

Q: Will this mod increase my load time a lot?
A: This mod may add a few seconds (or longer depending on how bad your computer is) when a save is first loaded as we need to find all to localization text in the game. Thankfully, we save what we found so subsequent first loads will not take as long. When ever the game updates, this load time increase may appear again as we will likely need to check for new text.

Q: How do I change my sim's preferred pronouns?
A: Click any sim and go to NeonOcean > Refer > Pronoun Settings. Select the "Pronouns" option in the dialog that appears to choose the sim's preferred pronouns.

Q: Will you add in a way for gender non-conforming sims to ask others to use their preferred pronouns?
A: I might add in a simple interaction to allow this but it wouldn't affect whether or not others use your sim's pronouns as there is no real way to determine who's talking. Just think of the sims world as one in which even your worst enemy respects your pronouns and everyone can use telepathy to determine what your preferred pronouns are.

Q: A set of pronouns I want my sim to use isn't in this mod, is there any way for me to add them?
A: Right now, there is no way for you to add in a custom pronoun set. But I do plan to add in a way soon.

Unsolvable problems

These issues exist because they would be very difficult or even impossible to fix.

  • They / Them Pronoun Issues
    •
    • "She's" and "he's" does not have a direct replacement for they / them pronouns. Both "they're" and "they've" work in different cases and its not really feasible for this mod to automatically detect when to use which. The only way I could fix this issue was to manually set which to use for each case. This means that Refer will ignore any instances of "she's" and "he's" that may occur in other mods not explicitly supported.

    • Using they / them pronoun may result in grammatically incorrect sentences in a few cases. There isn't any real way to rewrite the sentences on the fly or even detect if using they or them is grammatically incorrect. I recommend using a "neopronoun" instead if you really want to avoid this issue.

  • General Issues
    •
    • The game will occasionally use your sim's pronouns to refer to sims that don't actually exist in the world. For example, the game may make up a fake co-worker in the situational "should your sim do x or should they do y" dialogs and refer to them in the same way they do your sim.
